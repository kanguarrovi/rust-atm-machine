#  rust-atm-machine

A simple Rust project mimic of an ATM machine. The idea was learning to use Cargo for Rust projects.


Reference: https://doc.rust-lang.org/book/ch01-03-hello-cargo.html

To compile

    $ cargo build

To run in debug

    $ ./target/debug/rust-atm-machine


To run as a project

    $ cargo run
